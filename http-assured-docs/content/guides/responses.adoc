=== Available Assertions

All methods are static imports from `io.smallrye.httpassured.assertion.Assertions`:

[cols="1,2"]
|===
|Method |Description

|`isEqualTo(expected)`
|Value equals expected

|`isNotEqualTo(unexpected)`
|Value does not equal unexpected

|`isNull()`
|Value is null

|`isNotNull()`
|Value is not null

|`containsString(sub)`
|String contains substring

|`matchesPattern(regex)`
|String matches regular expression

|`hasSize(n)`
|Collection or array has n elements

|`contains(element)`
|Collection contains element

|`containsAll(e1, e2, ...)`
|Collection contains all elements

|`greaterThan(n)`
|Comparable value is greater than n

|`lessThan(n)`
|Comparable value is less than n

|`allMatch(predicate)`
|All elements match predicate

|`anyMatch(predicate)`
|At least one element matches predicate

|`satisfies(predicate)`
|Custom predicate check
|===
