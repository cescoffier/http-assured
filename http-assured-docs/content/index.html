http-assured

A modern Java HTTP test framework with a REST Assured-like DSL. No Groovy. No Hamcrest. No static global config.

Get Started
Pure Java

No Groovy dependency. No Hamcrest. Just Java 17+ with JUnit 5 assertions.

Pluggable Engine

SPI-based HTTP client. Vert.x by default, swap in JDK HttpClient or any custom engine.

Fault Injection

Built-in toxics for latency, connection resets, bandwidth limits. No external proxy needed.

WebSocket Testing

Session-based WebSocket API with blocking await and Mutiny Multi streaming.

Quick Example

HttpAssured client = HttpAssured.builder()
    .baseUri("http://localhost:8080")
    .build();

client.given()
    .header("Accept", "application/json")
.when()
    .get("/users")
.then()
    .statusCode(200)
    .body("name", Assertions::isNotNull);
